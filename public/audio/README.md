# Audio Files

This directory contains audio files for the interactive birthday invitation.

## Required Audio Files

### 1. background-music.mp3
- **Description**: Looping ambient background music (Indian fusion/curry restaurant atmosphere)
- **Duration**: 30-60 seconds (loops indefinitely)
- **Volume**: 30% (background, not overpowering)
- **Mood**: Warm, inviting, celebratory
- **Suggestions**:
  - Indian fusion music with soft sitar or tabla
  - Upbeat restaurant ambiance
  - Royalty-free sources: Pixabay, YouTube Audio Library, Freesound.org

### 2. spin-whoosh.mp3
- **Description**: Quick whoosh sound when user starts spinning the curry
- **Duration**: ~0.5 seconds
- **Volume**: 60%
- **Suggestions**:
  - Wind whoosh sound
  - Fast swoosh effect
  - Playful interaction sound

### 3. spin-sizzle.mp3
- **Description**: Cooking/sizzle sound at milestones (25%, 50%, 75%)
- **Duration**: ~1 second
- **Volume**: 60%
- **Suggestions**:
  - Food sizzling on pan
  - Curry cooking sound
  - Restaurant kitchen ambiance

### 4. explosion-reveal.mp3
- **Description**: Dramatic reveal sound when curry explodes and invitation appears
- **Duration**: ~2 seconds
- **Volume**: 80%
- **Suggestions**:
  - Magical sparkle/shimmer sound
  - Celebration fanfare
  - Confetti pop sound

### 5. button-click.mp3
- **Description**: UI button click feedback for RSVP button
- **Duration**: ~0.2 seconds
- **Volume**: 50%
- **Suggestions**:
  - Soft click sound
  - Button press feedback
  - Subtle UI interaction sound

## Current Status

Currently, placeholder empty MP3 files are used. These will not play any sound but won't cause errors either. The audio system is fully functional and will work once you add real audio files.

## How to Add Your Own Audio

1. Find or create your audio files (MP3 format)
2. Name them exactly as listed above
3. Replace the placeholder files in this directory
4. Reload the website to hear the audio
